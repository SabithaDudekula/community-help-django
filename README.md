# Community Help & Support Management System

This is a Django-based web application developed to help people connect with volunteers for community support. Users can post help requests, volunteer to help others, and track the status of requests.

ðŸ”— Live Project: https://community-help-django.onrender.com/

## Features

- User Registration and Login
- Post Help Requests
- View and Accept Requests
- Request Status Tracking (Open, In Progress, Resolved)
- User Dashboard
- Admin Panel for Management
- Simple and User-Friendly Interface

## How to Run This Project Locally
```bash
1. Clone the repository
git clone https://github.com/SabithaDudekula/community-help-django.git
2. Go to project folder
cd community-help-django
3. Create virtual environment
python -m venv env
env\Scripts\activate
4. Install requirements
pip install -r requirements.txt
5. Apply migrations
python manage.py migrate
6. Create admin user
python manage.py createsuperuser
7. Run the server
python manage.py runserver

Open in browser:
http://127.0.0.1:8000/

Register and login to the system
